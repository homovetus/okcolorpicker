About This Plugin

This plugin is designed to act like a built-in Photoshop tool, offering easy-to-use controls with no fancy function.

Mode

The colorpicker has two mode: Saturation-Lightness and Hue-Saturation. Right click the canvas to switch.

Source Code

This plugin is open-sourced and available for contributions. You can find the project here: https://github.com/homovetus/okpicker
